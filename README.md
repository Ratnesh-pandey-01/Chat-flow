# 📱 Flutter Chat App

A real-time chat application built with Flutter and Firebase featuring messaging, image sharing, voice messages, and push notifications.

---

## 🚀 Features

- 🔐 Login & Signup (Firebase Auth)
- 💬 Real-time Messaging
- 🖼️ Image Sharing
- 🎤 Voice Messages
- 🔔 Push Notifications (FCM)
- 🟢 Online/Offline Status
- 🌙 Dark Mode

---

## 🛠️ Tech Stack

- Flutter
- Firebase Auth
- Cloud Firestore
- Firebase Cloud Messaging (FCM)
- Cloudinary

---

## ⚙️ Setup Instructions

1. Clone the repo
2. Add your API keys in config.dart
3. Run the app
